# Bush League Bocce — App

React + Vite PWA. See the [root README](../README.md) for full project documentation.

## Development

```bash
npm install
npm run dev       # http://localhost:5173
```

Requires `app/.env.local` with Supabase credentials and admin password. See root README for details.

## Build

```bash
npm run build     # outputs to dist/
```

## Deploy

```bash
npx vercel --prod
```
